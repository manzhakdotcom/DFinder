Программа DFinder по сети подключается к файлам COR_Название-Участка.GDB "Журнала приказов и распоряжений" и получает список имен последних диспетчеров, которые провели циркулярный.

ВАЖНО!
 - Для работы программы нужно установить python 3.6.X и firebird 2. Путь к исполняемому файлу python.exe должен быть прописан в переменной среды PATH. Установить библиотеки future и fdb для python (зупустить run.bat).

=Настройка=
1. В файле settings.ini необходимо прописать данные для подключения к базе данных диспетчерского участка. 

=Принцип работы=
1. Запустить файл DFinder.exe.
2. Нажать на кнопку "Получить".
3. Для обновления списка диспетчеров нажмите кнопку "Обновить".

=Возможности=
1. Выыодит диспетчеров текущей и прошлой смен.
